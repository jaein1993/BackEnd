
# 로그인 화면 연습  

간단한 로그인 예제입니다.  
올바른 아이디/비밀번호를 입력하면 welcome 페이지로 넘어갑니다.  
틀린 아이디/비밀번호를 입력하면 다시 loginForm 페이지가 새로고침됩니다.  
로그인 후 welcome 페이지에서는 로그아웃을 할 수 있습니다.  
로그아웃 링크를 누르면 다시 loginForm 페이지로 돌아갑니다.  

- 올바른 아이디 : sky, sea, tree  
- 올바른 비밀번호 : abc1234
